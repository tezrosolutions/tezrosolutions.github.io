Including CSS, JavaScript files and Images (Not for Emails)

CSS and JavaScript files

  1. All JavaScript files will only be included in the footer.
  2. All CSS files will only be included in the header.
  3. No inline CSS and JavaScript is allowed. This will keep the code cleaner and manageable.
  4. Any third party CSS or JavaScript library will be added to a to a file like libraries.js or libraries.css. Separately including each library slows down the page load time and causes additional render blocking.
  5. Where possible, create one respective file for all custom CSS and Javascript code i.e. styles.css and scripts.js respectively.
  6. All custom Javascript code should be wrapped inside jquery document.ready function.

Images

  1. We will have three types of images i.e. 1x, 2x and 3x. Retina.js will be used to dynamically select the appropriate type for a screen.
  2. Use sprites and font icons whenever possible.
